Marketing Tool FRS State File Consolidator

A better way to
work with FRS data.

Drop an EPA FRS state zip file, and this tool unpacks it, joins the relational tables on REGISTRY_ID, and gives you one row per contact with facility, permit, program, and status data attached. Filter by environmental interest, program system, active status, affiliation type, geography, and more.
